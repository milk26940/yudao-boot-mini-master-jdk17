# 校园二手物品交易管理系统页面描述文档

## 一、文档说明

本系统页面设计分为普通用户端和管理员端两部分。为了更符合课程设计和毕业设计的写法，页面结构采用“功能集中、页面数量适中”的方式进行整理。

精简后的页面结构说明如下：

1. 普通用户端集中为 6 个页面
2. 管理员端集中为 6 个页面
3. 页面描述以论文展示和系统实现说明为导向
4. 后续实现时允许一个页面中包含多个功能模块

## 二、普通用户端页面

### 1. 首页

页面说明：
首页是系统的主界面，游客和登录用户都可以进入。该页面主要用于展示二手物品信息、公告通知、商品分类、搜索功能以及推荐信息。

主要功能：

1. 浏览商品信息
2. 按分类查看商品
3. 输入商品名称、品牌、价格等关键字进行搜索
4. 登录后查看系统推荐商品
5. 进入商品发布、求购发布和个人中心等功能模块

页面内容：

1. 导航栏
2. 搜索框
3. 商品分类区
4. 商品列表区
5. 公告通知区
6. 推荐商品区

### 2. 注册与登录页面

页面说明：
该页面用于普通用户完成注册申请和登录操作。用户注册时需填写基本信息并上传学生证照片，等待管理员审核；审核通过后方可登录系统。

主要功能：

1. 用户注册申请
2. 身份认证材料上传
3. 登录验证
4. 验证码校验

页面内容：

1. 姓名
2. 学号
3. 邮箱
4. 密码
5. 联系方式
6. 学生证上传
7. 验证码输入
8. 注册按钮
9. 登录按钮

### 3. 商品浏览与详情页面

页面说明：
该页面是系统核心业务页面，用于展示商品信息和商品详细内容。用户可查看商品的图片、价格、成色、描述及卖家联系方式。

主要功能：

1. 浏览商品
2. 搜索商品
3. 查看商品详情
4. 联系卖家
5. 在线购买
6. 发表评论

页面内容：

1. 商品列表
2. 商品图片
3. 价格
4. 分类
5. 关键字搜索栏
6. 商品详情信息
7. 评论区
8. 购买按钮
9. 联系方式展示

### 4. 商品发布与管理页面

页面说明：
该页面用于登录用户发布和管理自己的二手商品信息。

主要功能：

1. 发布商品信息
2. 上传商品图片
3. 编辑商品信息
4. 删除商品信息
5. 查看已发布商品状态

页面内容：

1. 商品名称
2. 价格
3. 成色
4. 品牌型号
5. 商品描述
6. 联系方式
7. 图片上传
8. 发布按钮
9. 修改按钮
10. 删除按钮

### 5. 求购信息页面

页面说明：
该页面用于用户发布和浏览求购信息，当用户未找到所需商品时可以发布自己的求购需求。

主要功能：

1. 发布求购信息
2. 查看求购信息
3. 修改自己发布的求购内容
4. 删除自己发布的求购内容

页面内容：

1. 求购物品名称
2. 品牌
3. 型号
4. 期望价格
5. 需求描述
6. 联系方式
7. 求购列表

### 6. 个人中心页面

页面说明：
个人中心页面用于用户统一管理个人资料及个人相关业务信息，是普通用户的综合管理页面。

主要功能：

1. 修改个人信息和密码
2. 查看我的商品
3. 查看我的求购
4. 查看订单信息
5. 查看评论与回复

页面内容：

1. 个人资料区
2. 密码修改区
3. 我的发布
4. 我的求购
5. 我的订单
6. 评论管理

## 三、管理员端页面

### 1. 管理员登录页面

页面说明：
该页面用于系统管理员登录后台管理系统，只有账号和密码验证通过后才能进入后台。

主要功能：

1. 管理员身份认证
2. 后台登录

页面内容：

1. 管理员账号输入框
2. 密码输入框
3. 登录按钮

### 2. 后台首页

页面说明：
后台首页是管理员登录后的主控制页面，用于展示系统整体运行情况及各管理模块入口。

主要功能：

1. 查看系统统计信息
2. 查看待审核注册信息
3. 查看商品总数
4. 查看用户总数
5. 查看公告数量
6. 进入各类管理功能页面

页面内容：

1. 统计面板
2. 快捷入口
3. 待办信息
4. 公告概览

### 3. 用户审核与管理页面

页面说明：
该页面用于管理员审核普通用户的注册申请，并统一管理所有用户信息。

主要功能：

1. 审核注册申请
2. 查看学生证信息
3. 通过或驳回申请
4. 管理用户资料
5. 删除异常账户

页面内容：

1. 待审核用户列表
2. 用户基本信息
3. 学生证照片
4. 审核按钮
5. 用户管理列表

### 4. 商品与求购信息管理页面

页面说明：
该页面用于管理员统一管理系统中的商品信息和求购信息，确保发布内容真实、规范。

主要功能：

1. 查看商品信息
2. 删除违规商品
3. 管理求购信息
4. 搜索和筛选信息

页面内容：

1. 商品列表
2. 求购列表
3. 发布者信息
4. 发布时间
5. 删除按钮
6. 查询条件

### 5. 交易与评论管理页面

页面说明：
该页面用于管理员查看系统交易记录及评论内容，维护平台正常交易秩序和交流环境。

主要功能：

1. 查看订单信息
2. 处理异常交易
3. 查看评论内容
4. 删除不当评论

页面内容：

1. 订单列表
2. 交易状态
3. 买卖双方信息
4. 评论列表
5. 评论处理按钮

### 6. 公告与系统管理页面

页面说明：
该页面用于管理员发布公告通知，并进行系统基础维护。

主要功能：

1. 发布公告
2. 修改公告
3. 删除公告
4. 维护商品分类
5. 查看系统通知记录

页面内容：

1. 公告编辑区
2. 公告列表
3. 分类管理区
4. 通知记录区

## 四、精简后的页面结构总结

### 1. 普通用户端

普通用户端可集中为 6 个页面：

1. 首页
2. 注册与登录页面
3. 商品浏览与详情页面
4. 商品发布与管理页面
5. 求购信息页面
6. 个人中心页面

### 2. 管理员端

管理员端可集中为 6 个页面：

1. 管理员登录页面
2. 后台首页
3. 用户审核与管理页面
4. 商品与求购信息管理页面
5. 交易与评论管理页面
6. 公告与系统管理页面

## 五、可直接写入论文的简要描述

本系统页面设计分为普通用户端和管理员端两部分。普通用户端主要包括首页、注册与登录、商品浏览与详情、商品发布与管理、求购信息以及个人中心等页面，能够满足用户进行商品浏览、发布、求购、交易和个人信息维护等需求。管理员端主要包括管理员登录、后台首页、用户审核与管理、商品与求购信息管理、交易与评论管理以及公告与系统管理等页面，用于完成系统的审核、监督和维护工作。精简后的页面结构更加清晰，能够突出系统核心业务功能，便于系统实现与论文撰写。
