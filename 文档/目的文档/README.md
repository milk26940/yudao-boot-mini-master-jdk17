# 目的文档目录说明

## 目录用途

本目录用于保存“校园二手物品交易管理系统”的长期设计文档、实现方案文档和开发计划文档。

这里的文档强调“可长期复用”和“指导实现”，不记录每天的临时状态。

## 维护原则

1. 长期方案、设计说明、结构性内容写入 `文档/目的文档`
2. 每日进度、待办事项、开发顺序写入 `文档/进度文档`
3. 新增文档时优先放到已有分类下，避免重复建同类文档
4. 方案文档要能直接支撑实现和论文整理
