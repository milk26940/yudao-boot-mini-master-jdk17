
菜单 SQL 导入与核对命令：

```bash
Get-Content sql/mysql/campus-second-trade-menu.sql | mysql -uroot -p123456 -D ruoyi-vue-pro

mysql --default-character-set=utf8mb4 -uroot -p123456 -D ruoyi-vue-pro -N -e "SELECT id,name,path,component FROM system_menu WHERE id IN (6100,6101,6102,6103,6104,6105,6106) ORDER BY id;"

```
