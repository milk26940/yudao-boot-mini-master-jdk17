# 当前开发顺序说明

当前项目采用这条推进顺序：

后端核心模块完成
-> 管理后台前端完成
-> 菜单权限与联调完成
-> 管理后台数据看板补齐
-> 普通用户端 MVP 第一批落地
-> 继续扩普通用户端后续页面
